# The impact score of farm based measures

This table contains the scores of crop rotation related farm measures
for Ecoregeling

## Usage

``` r
er_farm_measure
```

## Format

An object of class `data.table` (inherits from `data.frame`) with 45
rows and 5 columns.

## Details

- id:

  Unique id for each impact score related to crop rotation measures per
  environmental theme

- indicator:

  The environmental theme, choices: "soil", "water", "biodiversity",
  "climate" or "landscape"

- eco_id:

  Ecoregeling measure id

- description:

  Short description of the Ecoregeling measure

- er_score:

  The impact score of the Ecoregeling measure
