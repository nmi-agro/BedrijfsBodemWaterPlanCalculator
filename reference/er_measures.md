# The applicability of Ecoregeling Measures per crop type

This table contains the crops for which the Ecoregeling method counts
scores and rewards

## Usage

``` r
er_measures
```

## Format

An object of class `data.table` (inherits from `data.frame`) with 10450
rows and 3 columns.

## Details

- eco_id:

  The Ecoregeling measure id

- B_LU_BRP:

  The Dutch crop id from the BRP

- eco_app:

  A helper variable telling that the measure is applicable for that crop
