# Helper function to check, update and extend the table with measurements

Helper function to check, update and extend the table with measurements

## Usage

``` r
bbwp_check_meas(dt, eco = FALSE, score = TRUE)
```

## Arguments

- dt:

  (data.table) a data.table containing the bbwp measures with or without
  properties

- eco:

  (boolean) get only the measures for Ecoregeling (with a given eco_id)

- score:

  (boolean) is check done for scoring of subsets of full list of
  measures
