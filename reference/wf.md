# Helper function to weight and correct the risk and scores

Helper function to weight and correct the risk and scores

## Usage

``` r
wf(x, type = "indicators", penalty = TRUE)
```

## Arguments

- x:

  The risk or score value to be weighted

- type:

  Use the weighing function for indicators or score

- penalty:

  (boolean) the option to apply a penalty for high risk BBWP field
  indicators
